# Smart Study Planner (Java GUI Project)

## Overview

Smart Study Planner is a Java-based desktop application built using **Java Swing** that helps students manage their study tasks efficiently.
The application allows users to add tasks, set deadlines, assign priorities, track completion status, and save tasks for later use.

This project demonstrates concepts of:

* Object-Oriented Programming (OOP)
* Java Swing GUI
* File Handling
* Data Structures (ArrayList)
* Event-driven programming

---

## Problem Statement

Students often struggle to manage assignments, deadlines, and study plans effectively.
Many students forget tasks or lose track of their progress.

This project provides a simple tool to:

* Organize study tasks
* Track progress
* Manage priorities
* Save tasks automatically

---

## Features

* Add new study tasks
* Assign subject and deadline
* Set task priority (High / Medium / Low)
* Mark tasks as completed
* Delete tasks
* Task statistics (Total tasks and completed tasks)
* Automatic saving of tasks
* Load saved tasks on startup
* Priority-based color highlighting in task table

## How to Run the Project

1. Open the project in VS Code
2. Open terminal in the `src` folder
3. Compile the project:

```
javac *.java
```

4. Run the application:

```
java Main
```

---

## Example Workflow

1. Enter task details
2. Select priority
3. Click **Add Task**
4. Track progress
5. Tasks are saved automatically
